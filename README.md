QREADY AI HEMS PRO (Battery Simulator Edition)

ECHONET Lite規格を利用して、太陽光発電システム（QREADY-R59-1等）からリアルタイムに電力を取得し、蓄電池導入のシミュレーションを行うWebダッシュボードです。
🚀 主な機能

    リアルタイム監視: 発電量、消費電力、売買電状況を5秒ごとに更新。

    発電予測統合: Open-Meteo APIと連携し、設置地点の気候に基づいた未来の発電予想カーブを表示。

    蓄電池シミュレーター:

        日次モード: 今日の実際の買電量と売電余剰を比較し、夜間電力を自給するために必要な蓄電池容量をアドバイス。

        年次モード: 月ごとの実績データから、季節ごとの推奨蓄電池サイズをリスト表示。

    電気料金概算: 九州電力「電化でナイト・セレクト21」等の時間帯別単価に基づいた、その日の電気代をリアルタイム算出。

🛠 ネットワーク構成

    構成: 有線ルータ(無線) ↔ RE305(中継機) ↔ パワコン(QREADY-R59-1 有線接続)

    バックエンド: Python 3.x / Flask

    データベース: SQLite3 (エネルギーデータの自動蓄積)

    フロントエンド: Chart.js / Vanilla JS

📦 セットアップ
1. 依存ライブラリのインストール
Bash

pip install flask requests python-dotenv

2. 環境変数の設定

プロジェクトのルートディレクトリ（app.pyと同じ階層）に .env ファイルを作成し、以下の情報を設定してください。
Plaintext

HEMS_IP=192.168.0.xxx      # パワコンの固定IPアドレス
HEMS_LAT=33.xxx           # 設置場所の緯度
HEMS_LON=130.xxxx          # 設置場所の経度

3. 実行
Bash

python app.py

起動後、ブラウザで http://localhost:8000 にアクセスしてください。
📊 表示モードの解説

    前後1時間: 現在のリアルタイムな動きと、直近1時間の発電予測を詳細に確認できます。

    今日: 0時からの推移グラフ。リスト表示では「本日の推奨蓄電池容量」が自動計算されます。

    今年: 月別の統計データ。季節ごとの発電量の差と、それに基づいた最適な電池サイズを比較できます。

⚠️ 注意事項

    本ツールは個人の学習およびエネルギー最適化を目的としています。

    ECHONET Liteの仕様により、ネットワーク環境によってはデータ取得に失敗する場合があります。
